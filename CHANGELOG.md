## [1.0.6] - 2025-11-17

### أُضيف - الأسبوع 3 (85%)

#### واجهة برمجية REST API
- 8 نقاط وصول RESTful
- المصادقة عبر WordPress cookies/app passwords
- دعم الترقيم (pagination)
- البحث والفلترة في API
- تحديد المعدل
- صفحة توثيق API في لوحة الإدارة

**نقاط الوصول:**
- `GET /wp-json/aqop/v1/leads` - قائمة العملاء
- `GET /wp-json/aqop/v1/leads/{id}` - عميل واحد
- `POST /wp-json/aqop/v1/leads` - إنشاء عميل
- `PUT /wp-json/aqop/v1/leads/{id}` - تحديث عميل
- `DELETE /wp-json/aqop/v1/leads/{id}` - حذف عميل
- `GET /wp-json/aqop/v1/leads/statuses` - الحالات
- `GET /wp-json/aqop/v1/leads/countries` - الدول
- `GET /wp-json/aqop/v1/leads/sources` - المصادر
